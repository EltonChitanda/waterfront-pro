# Waterfront Pro — Business Website

A fully responsive business website built for **Waterfront Pro (Pty) Ltd**, a construction and plumbing company based in Vaalwater, Limpopo. This was a freelance project built from scratch using HTML, CSS, JavaScript and Bootstrap 5.

🌐 **Live Site:** [waterfront-pro.netlify.app](https://waterfrontpro.co.za)

---

## About The Project

Waterfront Pro needed a professional online presence to showcase their services and make it easy for clients to get in touch. The site was designed mobile-first since the majority of their clients are on mobile devices.

---

## Built With

- HTML5
- CSS3
- JavaScript
- Bootstrap 5.3.8
- Formspree (contact form)
- Netlify (hosting)
- Git & GitHub (version control)

---

## Features

- Fully responsive — mobile first design
- Sticky navbar with active link highlighting
- Auto-playing hero carousel with dark overlay
- Services preview section with image grid
- Coverage map showing Limpopo and Gauteng
- Filtered photo gallery (Building, Plumbing, Thatch, Cemecrete, Maintenance)
- Contact form powered by Formspree
- Floating WhatsApp button
- Scroll to top button
- Google Search Console integration
- Favicon
- SEO meta tags

## Running Locally

```bash
git clone https://github.com/EltonChitanda/waterfront-pro.git
cd waterfront-pro
```

Open `index.html` with Live Server in VS Code.

---

## Deployment

The site is deployed on **Netlify** with automatic deploys connected to this GitHub repository. Every push to `main` triggers a new deployment.
